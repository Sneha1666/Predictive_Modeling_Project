# Predictive Modeling Using Machine Learning

## Overview

This project demonstrates Predictive Modeling using Machine Learning. A Random Forest Classifier is trained on the Iris dataset to predict flower species based on their features. The model is evaluated using accuracy score, classification report, confusion matrix, and ROC curve.

## Objectives

* Apply supervised machine learning techniques.
* Train and test a predictive model.
* Evaluate model performance.
* Visualize results using confusion matrix and ROC curve.

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Matplotlib

## Dataset

The Iris dataset is used in this project. It contains measurements of iris flowers and their corresponding species.

Features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

Target:

* Iris Species

## Algorithm Used

Random Forest Classifier

## Project Workflow

1. Load the Iris dataset.
2. Split the dataset into training and testing sets.
3. Train the Random Forest model.
4. Predict outcomes on the test dataset.
5. Evaluate model performance using:

   * Accuracy Score
   * Classification Report
   * Confusion Matrix
   * ROC Curve
6. Visualize the results.

## Results

* Achieved high prediction accuracy.
* Generated confusion matrix for classification analysis.
* Generated ROC curve for performance evaluation.
